# 01-visual-studio-copilot

# Komentteja

-Kuvissa voisi olla linkit isompiin kuviin, koska kuvissa näyttäisi olevan hyvin resoluutiota.

- Jos copilot ei näy editorissa:Code--> Preferences--> Settings Search settings:Kirjoita Copilot--> valitse "Chat"--> Ota ruksi pois kohdasta: Chat:Disable Ai Features
- Copilot-paneelin avaava työkalu ei ole materiaalin kuvan osoittamassa paikassa. Lisäksi työkalun ikonin ulkonäkö voi vaihdella. Tämän vuoksi olisi hyvä kertoa pikanäppäimet Copiöot-paneelin esiin saamiseksi. MacOs: <command + option + B>, Windows:ehkä <Ctrl + Alt + B>
